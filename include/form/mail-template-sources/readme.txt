Use files in this folder to build a complete mail template with foundation emails,
using the Zurb Responsive Email Inliner:
https://foundation.zurb.com/emails/inliner-v2.html

 - johnnyword_source_mail.html: mail html source: edit and paste in Foundation Email Inliner to build ../johnnyword.mail.html
 - johnnyword_source_mail.css: mail css: edit and paste in Foundation Email Inliner to build ../johnnyword.mail.html

 - index.html: foundation base template
 - css/: base foundation emails css files
 - templates/: foundation emails templates
